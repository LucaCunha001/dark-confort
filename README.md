# Dark Confort Theme for Visual Studio Code

Dark Confort is a minimalistic dark theme for VS Code, designed for developers who love **high-contrast, nearly black backgrounds** and subtle color accents. It comes in **two variants**:

- **Dark Confort All Black** – ultra-dark, almost pure black for the editor, sidebars, and tabs.
- **Dark Confort Red** – same dark base with **red accents** for active elements, selections, and highlights.

---

## Features

- Deep black backgrounds for editor, sidebar, activity bar, and title bar.
- Minimalist and comfortable color palette for long coding sessions.
- Semantic highlighting enabled, letting VS Code color your code intelligently.
- Red accents (optional) for active tabs, selection, and focus states.
- Clean interface with improved legibility and reduced eye strain.

## Installation

1. Open Visual Studio Code.
2. Go to the **Extensions** panel (`Ctrl+Shift+X` or `Cmd+Shift+X` on macOS).
3. Search for `Dark Confort` and install.
4. Open the **Command Palette** (`Ctrl+Shift+P` or `Cmd+Shift+P`), then type `Color Theme` → `Preferences: Color Theme`.
5. Choose either **Dark Confort All Black** or **Dark Confort Red**.
